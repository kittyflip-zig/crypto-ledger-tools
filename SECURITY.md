# Security Policy

## Supported Versions

This repository is an OSS publication candidate. Security review applies to the current candidate files only.

## Do Not Submit Sensitive Data

Do not submit:

- API keys
- Access tokens
- Passwords
- Cookies
- Webhook URLs
- Real transaction CSV files
- Real wallet address values
- Personal identifiers

## Reporting

If you find a security issue in this candidate, report it privately to the repository maintainer before public disclosure.

Do not include sensitive proof data in an issue, pull request, discussion, or sample file.
